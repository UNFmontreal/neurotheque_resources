# CLI Reference

The `neuroflow` command-line interface provides several commands to help you manage your pipelines.

## `neuroflow run`

Runs a pipeline from a given YAML configuration file.

```bash
neuroflow run [OPTIONS] CONFIG_FILE
```

**Arguments**:

*   `CONFIG_FILE`: The path to the YAML configuration file. [required]

## `neuroflow new-config`

Launches an interactive wizard to create a new configuration file.

```bash
neuroflow new-config
```
