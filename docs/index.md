# Welcome to Neuroflow

**Neuroflow** is a Python-based pipeline for processing neurophysiological data, with a focus on EEG recordings. It provides a flexible, step-based architecture for building and running custom processing workflows.

![Pipeline Animation](images/pipeline_animation_v2.gif)

## Key Features

- **Modular Architecture**: Build custom pipelines by combining individual processing steps.
- **BIDS Compliant**: Organizes data and derivatives according to the Brain Imaging Data Structure (BIDS) standard.
- **Automated Reporting**: Generate detailed HTML reports with figures and metrics from your pipeline runs.
- **Command-Line Interface**: Control your pipelines from the terminal with the `neuroflow` CLI.
- **Reproducibility**: Ensure your analysis is reproducible with version-pinned dependencies and containerized environments.
